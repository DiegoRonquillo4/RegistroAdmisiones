# RegistroAdmisiones
Registro de estudiantes que aprovaron el Pre de Admisiones.
Adjuntar datos de:
- Matricula
- Nombres Completos
- Apellidos Completos
- Edad(al terminar el pre)
- ID de la carrera
- Correo electronico
